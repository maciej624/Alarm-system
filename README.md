# Vision Alarm System

Real-time human detection system using YOLOv8 and OpenCV,
with automatic recording and Arduino integration.

![demo](presentation.gif)

## What it does

- Detects people in real time using YOLOv8 neural network
- Triggers alarm when person is detected
- Automatically records video to file
- Saves snapshot photo on each alarm
- Sends signal to Arduino (buzzer / LED)


## Tech stack

- Python 3.10
- YOLOv8 (ultralytics)
- OpenCV
- PySerial
- Arduino (optional)
